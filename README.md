# **WoWHeadProject**
WoWHead website based app with database of all objects from World of Warcraft

# **Цель работы:**
1. Создание приложения позволяющего увидеть данные о нужном обьекте из видеоигры World of Warcraft.

# **Задачи:**
1. Поиск предметов World of Warcraft из базы данных 
2. Добавление новых предметов в базу данных (По мере добавления нового контента в игру)
3. Удаление старых предметов из базы данных (По мере удаления устаревшего контента из игры)
